・制作プログラム「3DMC4L.amxd」

本システムはMax for Liveを搭載したAbleton Live上で動作するコンプレッサーです。

使用方法
1.Ableton Liveのユーザーライブラリ内にあるMax Audio Effectフォルダに「3DMC4L.amxd」を移動させる

2.3Dconnexion社の SpaceExplorerまたはSpaceNavigatorを接続する

3.Liveを起動すると、カテゴリセクションのMax for Liveフォルダ内にあるMax Audio Effectフォルダに3DMC4L.amxdがあるため、ダブルクリックやドラッグにより任意のオーディオトラックに挿入する

4.操作パネルが表示されたら、DeviceSelectメニューから接続した3Dマウスを選択する


操作に関しては、3Dマウスのキャップの押し込み・引っ張りでスレッショルドがコントロールでき、左右のロテートでメイクアップがコントロールできるようになっています。
